Game Project to test the engine functionalities

How to install:
-Download source and keep the map structure intact.
-Download the source of engine as well (see this README.md to install)

-Create a new project in visual studio:
-Add directx10 include and libs to VC++ directories.
-Add Engine headers as additional include directory.
-Add Engine library as library directories.
-Add This source as additional include directory.

Compile and run

(optional add the engine solution to project and let game depend on engine)
